- zártabb egység
- a klasszikus formálás legkisebb alapvető zenei egysége
- egy periódus -> egytagú forma
	- rendszerint 8 ütem (4+4)
	- a b
- két periódus -> kéttagú forma
	- A B
	- 2x kérdés-felelet
- visszatérés -> háromtagú forma
