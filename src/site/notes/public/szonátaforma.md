#

klasszikus szonáta
- uralkodó a klasszikus zenében
- kétrészes szonáták másofik fele sokszor kibővült kidolg. szerű résszel -> 3 részesség felé vezet

| A     | B                       |
| ----- | ----------------------- |
| T - D | D - T                   |
| I - V | V- I                    |
| D vagy párhuzamos hangnemben végződik      | Alaphangnemben végződik |

### Több témájúság
- 1. téma = főtéma
	- (átvezetés)
- 2. téma = melléktéma
- 3. téma = zárótéma
